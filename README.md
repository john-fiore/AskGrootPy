# Ask Groot
This is a little app I wrote in Python in under 10 minutes because I was bored.

It's inspired by the [AskGroot.com](https://www.askgroot.com/) website, but mostly [this scene](https://www.youtube.com/watch?v=vBEGFgRur-0) from the Wreck-it-Ralph sequel.
